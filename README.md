# Snakeladder-Areana – Online Multiplayer Snakes & Ladders

SocketSnakes is a real-time multiplayer implementation of the classic **Snakes and Ladders** board game built with **React, Node.js, and WebSockets (ws)**.

Players can join the same game room and compete with friends online while the server synchronizes game state, dice rolls, and player movements in real time.

---

##  Features

- Real-time multiplayer gameplay
- Dice roll system
- Snake penalties
- Ladder boosts
- Live game updates using WebSockets
- Interactive React game board
- Turn-based player system
- Server-side game logic

---

##  Tech Stack

### Frontend
- React
- HTML5 / CSS3
- JavaScript

### Backend
- Node.js
- WebSockets (`ws`)

### Communication
- Real-time bidirectional communication using WebSockets
